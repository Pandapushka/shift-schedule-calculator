# Инструкция по запуску MVP приложения "Калькулятор графика смен"

## Обзор
Это MVP версия приложения для расчета графика смен. Бэкенд на ASP.NET Core с SQLite, фронтенд на Vue.js без сборки.

## Требования
- .NET 10.0 SDK (установлен в `~/.dotnet`)
- Браузер с поддержкой JavaScript (Chrome, Firefox и т.д.)
- macOS (текущая ОС)

## Структура проекта
```
Новая папка/
├── backend/          # ASP.NET Core бэкенд
│   └── src/
│       ├── WebApi/   # API проект
│       ├── Application/
│       ├── Domain/
│       └── Infrastructure/
├── frontend/         # Vue.js фронтенд
│   └── index.html    # Одностраничное приложение
├── technical-task.txt      # Полное ТЗ
└── mvp-technical-task.txt  # MVP ТЗ
```

## Запуск бэкенда

1. Откройте терминал в папке проекта:
   ```
   cd '/Users/panda.pushka/Desktop/Новая папка'
   ```

2. Перейдите в папку backend:
   ```
   cd backend
   ```

3. Установите переменную PATH для .NET (если не установлена глобально):
   ```
   export PATH="$PATH:$HOME/.dotnet"
   ```

4. Запустите бэкенд:
   ```
   dotnet run --project src/WebApi
   ```

5. Дождитесь сообщения "Now listening on: http://localhost:5000". Бэкенд запущен.

   - Если порт 5000 занят, выполните:
     ```
     lsof -ti:5000 | xargs kill -9
     ```
     Затем повторите шаг 4.

6. Проверьте работу API (опционально):
   ```
   curl -X POST http://localhost:5000/api/shiftschedule/calculate \
        -H "Content-Type: application/json" \
        -d '{"startDate":"2026-01-01","workDays":2,"offDays":2,"hoursPerShift":11,"months":1,"nightFirst":false,"title":"Test"}'
   ```
   Должен вернуться JSON с графиком.

## Запуск фронтенда

1. Откройте браузер (Chrome, Firefox и т.д.).

2. В адресной строке введите:
   ```
   file:///Users/panda.pushka/Desktop/Новая папка/frontend/index.html
   ```

3. Страница загрузится. Если Bootstrap CSS не загрузился (из-за отсутствия интернета), функциональность все равно работает.

## Тестирование

1. Убедитесь, что бэкенд запущен (терминал показывает "Application started").

2. В браузере заполните форму:
   - Дата первого рабочего дня: 2026-01-01
   - Рабочих дней: 2
   - Выходных дней: 2
   - Часов в смене: 11
   - Месяцев для расчета: 1
   - Название графика: (опционально)

3. Нажмите "Рассчитать график".

4. Результат отобразится ниже в виде календаря с днями работы/отдыха и итоговыми подсчетами.

## Остановка

- Бэкенд: В терминале нажмите Ctrl+C
- Фронтенд: Просто закройте вкладку браузера

## Возможные проблемы

- **"Failed to fetch"**: Бэкенд не запущен или CORS ошибка. Проверьте, что бэкенд слушает порт 5000.
- **Порт занят**: Убейте процесс на порту 5000 командой выше.
- **.NET не найден**: Убедитесь, что export PATH выполнен.
- **База данных**: SQLite файл создается автоматически в backend/src/WebApi/shiftschedule.db
